# 🔢 Day 11 – Fibonacci Series  

## 🎯 Problem Statement  
Write a program to print the Fibonacci series up to a given limit using a **do-while loop**.  

---

## 📝 Sample Input & Output  

### Input  
Enter the Limit: 5  

### Output  
1  
1  
2  
3  
5  

### Input  
Enter the Limit: 7  

### Output  
1  
1  
2  
3  
5  
8  
13
