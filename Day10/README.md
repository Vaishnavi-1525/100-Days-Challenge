# 🔢 Day 10 – Factorial of a Number  

## 🎯 Problem Statement  
Write a program to calculate the factorial of a given number.  

---

## 📝 Sample Input & Output  

### Input  
Enter a Number: 5  
### Output  
Factorial of 5 is 120  

### Input  
Enter a Number: 0  
### Output  
Factorial of 0 is 1
