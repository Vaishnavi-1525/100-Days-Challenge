# 🔤 Day 03 – ASCII Value of a Character  

## 🎯 Problem Statement  
Write a program to find the ASCII value of a given character.  

---

## 📝 Sample Input & Output  

### Input  
Enter a Character: A  
### Output  
The ASCII value of 'A' is: 65  

### Input  
Enter a Character: a  
### Output  
The ASCII value of 'a' is: 97  

### Input  
Enter a Character: 5  
### Output  
The ASCII value of '5' is: 53  

### Input  
Enter a Character: @  
### Output  
The ASCII value of '@' is: 64
